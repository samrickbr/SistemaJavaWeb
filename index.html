Integração de html com css.

Software "Notepad++"

Utilizado o software "Notepad++" para a criação da página e integração do "HTML" e "CSS".

Na criação foram utilizadas "tags" comuns de HTML, alguns exemplos:

  • HEAD - Para o cabeçãho
  • BODY - Onde é inserido todo o conteúdo do corpo do site.
  • HEADER - Introdução ao conteúdo.
  • DIV - Elemento de divisão para agrupar itens na página.
